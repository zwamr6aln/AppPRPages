LockInNote

Widget-fokussierte Anwendung für die Erstellung von Notizen!

Überblick.

Dies ist eine Widget-spezifische Notizen-Anwendung für iPhone/iPad/Mac/Apple Watch.

Wenn Sie eine Notiz machen wollen, können Sie schnell eine Notiz über das Widget schreiben und die Notiz jederzeit auf dem Widget überprüfen!

Darüber hinaus kann das Design der Notiz auf dem Widget flexibel angepasst werden, so dass es für eine Vielzahl von Zwecken verwendet werden kann.

Multiplattform-Unterstützung

  • iPhone: Startbildschirm, Sperrbildschirm, Standby (iOS 17)
  • iPad: Startbildschirm, Sperrbildschirm (iPad OS 17)
  • Mac: Benachrichtigungszentrale, Schreibtisch (macOS 14)
  • Apple Watch: Komplikationen, Smart Stack (watchOS 10)
