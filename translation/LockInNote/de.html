LockInNote

Eine App für Notizen, die dem neuen iOS16 Lock Screen Widget Feature gewidmet ist, das im Herbst 2022 veröffentlicht wird!

Überblick.

Dies ist eine Notizen-App, die sich auf Sperrbildschirm-Widgets spezialisiert hat. Nur für das iPhone.

Wenn Sie sich eine Notiz machen wollen, können Sie diese schnell auf dem Sperrbildschirm schreiben und die Notiz jederzeit auf dem Sperrbildschirm überprüfen!

Das Design der Notiz auf dem Sperrbildschirm kann flexibel angepasst werden, so dass Sie es entsprechend Ihren verschiedenen Zwecken beim "Hinterlassen von Notizen" und "Anzeigen von Text auf dem Sperrbildschirm" einsetzen können.

Durch Antippen des Widgets wird sofort die Tastatur zum Bearbeiten der Notiz aufgerufen.

Passen Sie Schriftgröße, Design und Dicke an.

Leicht verständliche Icons, wenn kein Text eingegeben wird

  • Jedes URL-Schema aufrufen, das mit Text funktioniert
