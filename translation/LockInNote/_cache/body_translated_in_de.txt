<h2 id="概要"> Überblick.</h2>
<p> Dies ist eine Widget-spezifische Notizen-Anwendung für iPhone/iPad/Mac/Apple Watch.</p>
<p> Wenn Sie eine Notiz machen wollen, können Sie schnell eine Notiz über das Widget schreiben und die Notiz jederzeit auf dem Widget überprüfen!</p>
<p> Darüber hinaus kann das Design der Notiz auf dem Widget flexibel angepasst werden, so dass es für eine Vielzahl von Zwecken verwendet werden kann.</p>
<h3 id="マルチプラットフォーム対応"> Multiplattform-Unterstützung</h3>
<ul>
<li> iPhone: Startbildschirm, Sperrbildschirm, Standby (iOS 17)</li>
<li> iPad: Startbildschirm, Sperrbildschirm (iPad OS 17)</li>
<li> Mac: Benachrichtigungszentrale, Schreibtisch (macOS 14)</li>
<li> Apple Watch: Komplikationen, Smart Stack (watchOS 10)</li>
</ul>
