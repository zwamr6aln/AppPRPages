Plain将棋盤 / Plain Shogi Board

Самое простое приложение для доски сёги

Обзор.

Это обычная доска для игры в сёги с фигурами, которые можно свободно перемещать. Приложение для iPhone и iPad.

Простое приложение для доски сёги, выполненное в стандартном дизайне iOS.

Размер и толщина текста изменяются в зависимости от системных настроек ОС.

Вы можете удалить или добавить любой фрагмент.

Пьесы могут быть переведены в английскую нотацию.

Экспорт и импорт платы в виде текстовых данных.

Поддерживается функция SharePlay.

Вы можете играть в сёги со своими друзьями через FaceTime.

Функция SharePlay предназначена для различных сценариев использования, например, для решения проблемы Tsume-Shogi с большой группой друзей, а также для "простой игры между двумя людьми".
