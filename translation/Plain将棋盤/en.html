The Most Plain Shogi Board App

Overview

This is just a plain Shogi board with pieces you can move freely. iPhone and iPad app.

It is a simple Shogi board app that follows the standard iOS design.

The size and thickness of the text changes according to the OS system settings.

You can delete or add any piece.

Pieces can be changed to English.

Export/import board as text data.

SharePlay is supported.

You can play Shogi with your friends over FaceTime.

The SharePlay function is not only for "a simple game between two players" but also for various use cases such as "solving Tsume-Shogi with a large number of people".

Text Export/Import

The board can be exported and imported as text data.
