TapTemperature

Enregistrez votre température corporelle dans l'application Apple "Health Care" le plus rapidement possible.

Vue d'ensemble.

C'est la façon la plus rapide (manuelle) d'enregistrer les données relatives à votre température corporelle dans l'application Apple "Health Care" préinstallée sur votre iPhone.

Utilisateurs prévus / cas d'utilisation

  • Ne dispose pas d'un thermomètre intelligent.
  • Ils mesurent quotidiennement leur température corporelle (ou température basale) à l'aide d'un thermomètre.
  • Envisager d'enregistrer manuellement les résultats des mesures dans l'application "Health Care" immédiatement après chaque mesure avec un thermomètre ordinaire.

Option

  • Unité
    • °C Celsius
    • ℉ Fahrenheit
  • Mode de température corporelle basale
  • Mode deux décimales
  • Fonction permettant de compléter automatiquement le processus par la simple saisie de chiffres.

Autre.

  • Lancement d'une seule touche de l'application Apple Health Care à partir de cette application.
  • Fonction d'historique dans l'appareil pour le contrôle des opérations et la sauvegarde simple.
  • Fonction permettant d'annuler l'enregistrement d'une seule touche immédiatement après l'enregistrement des données.

Attention.

Cette application ne peut pas lire, afficher ou gérer les données antérieures de l'application Soins de santé. Cette application est destinée uniquement à l'enregistrement des données dans l'application "Soins de santé". Veuillez vérifier les données enregistrées dans l'application "Health Care".

Le contexte.

Les gens mesurent fréquemment leur température corporelle (et leur température basale) sur une base quotidienne à l'aide d'un thermomètre. De nombreux utilisateurs d'iPhone enregistrent leurs données de température (ou de température basale) sur l'application "Health Care".

La meilleure solution consiste à utiliser un thermomètre intelligent qui fonctionne avec l'application Health Care et enregistre automatiquement les mesures, mais ces appareils peuvent être chers et inaccessibles. Les données peuvent également être enregistrées manuellement sur l'application "Health Care", mais celle-ci n'est malheureusement pas conçue pour l'enregistrement manuel de routine des données de mesure. Par conséquent, la saisie manuelle de données qui se produisent en continu sur une base quotidienne, comme les mesures de la température corporelle, peut être une expérience très longue et stressante.

Cette application a été développée pour résoudre ces problèmes.
