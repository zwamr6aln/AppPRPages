TapTemperature

Daftarkan suhu tubuh Anda di aplikasi Apple 'Health Care' secepat mungkin.

Ikhtisar.

Ini adalah cara tercepat (manual) untuk mendaftarkan data suhu tubuh Anda di aplikasi Apple 'Health Care' yang sudah terpasang di iPhone Anda.

Pengguna yang dituju / kasus penggunaan

  • Tidak memiliki termometer pintar.
  • Mengukur suhu tubuh (atau suhu tubuh basal) dengan termometer setiap hari.
  • Mempertimbangkan untuk mendaftarkan hasil pengukuran secara manual di aplikasi 'Perawatan Kesehatan' segera setelah setiap pengukuran dengan termometer biasa.

Opsi.

  • Mode suhu tubuh basal
  • Mode dua tempat desimal
  • Berfungsi untuk menyelesaikan proses secara otomatis hanya dengan memasukkan angka

Lainnya.

  • Dapat juga digunakan dengan Apple Watch.
  • Peluncuran aplikasi Apple Health Care dengan satu ketukan dari dalam aplikasi ini.
  • Pembatalan pendaftaran dengan satu ketukan segera setelah pendaftaran data.
  • Unit
    • °C Celcius
    • ℉ Fahrenheit

Catatan.

Aplikasi ini tidak dapat membaca, melihat, atau mengelola data sebelumnya pada aplikasi 'Health Care'. Aplikasi ini hanya ditujukan untuk mendaftarkan data di aplikasi 'Health Care'. Silakan periksa data yang didaftarkan pada aplikasi 'Health Care'.

Latar belakang.

Orang sering mengukur suhu tubuh mereka (dan suhu tubuh basal) setiap hari dengan menggunakan termometer. Banyak pengguna iPhone mendaftarkan data suhu (atau suhu tubuh basal) mereka pada aplikasi 'Health Care'.

Solusi terbaik adalah dengan menggunakan termometer pintar yang bekerja dengan aplikasi Health Care dan secara otomatis menyimpan pengukuran, tetapi ini bisa jadi mahal dan tidak dapat diakses. Data juga dapat didaftarkan secara manual pada aplikasi 'Health Care', tetapi sayangnya aplikasi 'Health Care' tidak dirancang untuk pendaftaran data pengukuran secara manual secara rutin. Oleh karena itu, memasukkan data secara manual yang terjadi secara terus menerus setiap hari, seperti pengukuran suhu tubuh, dapat menjadi pengalaman yang sangat menyita waktu dan membuat stres.

Aplikasi ini dikembangkan untuk mengatasi masalah tersebut.
