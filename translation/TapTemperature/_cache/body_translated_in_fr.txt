<h2 id="概要"> Aperçu.</h2>
<p> Il s'agit de la méthode la plus rapide (manuelle) pour enregistrer les données relatives à votre température corporelle dans l'application Apple "Health Care" préinstallée sur votre iPhone.</p>
<h3 id="想定ユーザー-ユースケース"> Utilisateurs visés / cas d'utilisation</h3>
<ul>
<li> Ne dispose pas d'un thermomètre intelligent.</li>
<li> Mesure quotidiennement la température corporelle (ou température basale) avec un thermomètre.</li>
<li> Envisage d'enregistrer manuellement les résultats des mesures dans l'application 'Healthcare' immédiatement après chaque mesure avec un thermomètre ordinaire.</li>
</ul>
<h3 id="オプション"> Option.</h3>
<ul>
<li> Mode température corporelle basale</li>
<li> Mode deux décimales</li>
<li> Fonction permettant de compléter automatiquement le processus en saisissant simplement les chiffres.</li>
</ul>
<h3 id="その他"> Autre.</h3>
<ul>
<li> Peut également être utilisé avec l'Apple Watch.</li>
<li> Lancement d'une seule touche de l'application Apple Health Care à partir de cette application.</li>
<li> Annulation de l'enregistrement d'une seule touche immédiatement après l'enregistrement des données.</li>
<li> Unités<ul>
<li> °C Celsius</li>
<li> ℉ Fahrenheit</li>
</ul></li>
</ul>
<h3 id="注意"> Remarques .</h3>
<p> Cette application ne peut pas lire, afficher ou gérer les données antérieures sur l'application 'Health Care'. Cette application est destinée uniquement à l'enregistrement des données dans l'application 'Health Care'. Veuillez vérifier les données enregistrées dans l'application 'Health Care'.</p>
<h3 id="背景"> Contexte.</h3>
<p> Les gens mesurent fréquemment leur température corporelle (et leur température corporelle basale) au quotidien à l'aide d'un thermomètre. De nombreux utilisateurs d'iPhone enregistrent leurs données de température (ou de température basale) sur l'application "Health Care".</p>
<p> La meilleure solution consiste à utiliser un thermomètre intelligent qui fonctionne avec l'application Health Care et enregistre automatiquement les mesures, mais ces appareils peuvent être chers et inaccessibles. Les données peuvent également être enregistrées manuellement sur l'application "Health Care", mais celle-ci n'est malheureusement pas conçue pour l'enregistrement manuel de routine des données de mesure. Par conséquent, la saisie manuelle de données qui se produisent en permanence sur une base quotidienne, comme les mesures de la température corporelle, peut être une expérience très longue et stressante.</p>
<p> Cette application a été développée pour résoudre ces problèmes.</p>
