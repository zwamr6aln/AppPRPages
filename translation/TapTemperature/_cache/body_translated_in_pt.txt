<h2 id="概要"> Visão geral.</h2>
<p> Esta é a forma mais rápida (manual) de registar os seus dados de temperatura corporal na aplicação 'Cuidados de Saúde' da Apple pré-instalada no seu iPhone.</p>
<h3 id="想定ユーザー-ユースケース"> Utilizadores pretendidos / casos de utilização</h3>
<ul>
<li> Não tem um termómetro inteligente.</li>
<li> Mede a temperatura corporal (ou temperatura corporal basal) com um termómetro numa base diária.</li>
<li> Considerando o registo manual dos resultados de medição na aplicação "Cuidados de saúde" imediatamente após cada medição com um termómetro regular.</li>
</ul>
<h3 id="オプション"> Opção.</h3>
<ul>
<li> Modo de temperatura corporal basal</li>
<li> Modo de duas casas decimais</li>
<li> Função para completar automaticamente o processo através da simples introdução de números</li>
</ul>
<h3 id="その他"> Outros.</h3>
<ul>
<li> Também pode ser usado com o Apple Watch.</li>
<li> Lançamento da aplicação Apple Health Care a partir desta aplicação.</li>
<li> Cancelamento do registo num só toque imediatamente após o registo dos dados.</li>
<li> Unidades<ul>
<li> °C Celsius</li>
<li> ℉ Fahrenheit</li>
</ul></li>
</ul>
<h3 id="注意"> Notas.</h3>
<p> Esta aplicação não pode ler, visualizar ou gerir dados passados na aplicação "Cuidados de Saúde". Este aplicativo destina-se apenas ao registo de dados no aplicativo 'Cuidados de Saúde'. Por favor, verificar os dados registados na aplicação 'Cuidados de Saúde'.</p>
<h3 id="背景"> Antecedentes.</h3>
<p> As pessoas medem frequentemente a sua temperatura corporal (e temperatura corporal basal) numa base diária utilizando um termómetro. Muitos utilizadores de iPhone registam os seus dados de temperatura (ou temperatura corporal basal) na aplicação 'Cuidados de Saúde'.</p>
<p> A melhor solução é utilizar um termómetro inteligente que funcione com a aplicação 'Cuidados de Saúde' e armazene automaticamente as medições, mas estas podem ser dispendiosas e inacessíveis. Os dados também podem ser registados manualmente na aplicação 'Cuidados de Saúde', mas infelizmente a aplicação 'Cuidados de Saúde' não foi concebida para o registo manual de rotina de dados de medição. Portanto, a introdução manual de dados que ocorrem continuamente numa base diária, tais como medições da temperatura corporal, pode ser uma experiência muito demorada e estressante.</p>
<p> Esta aplicação foi desenvolvida para resolver tais problemas.</p>
