<h2 id="概要"> Ikhtisar.</h2>
<p> Ini adalah cara tercepat (manual) untuk mendaftarkan data suhu tubuh Anda di aplikasi Apple 'Health Care' yang sudah terpasang di iPhone Anda.</p>
<h3 id="想定ユーザー-ユースケース"> Pengguna yang dituju / kasus penggunaan</h3>
<ul>
<li> Tidak memiliki termometer pintar.</li>
<li> Mengukur suhu tubuh (atau suhu tubuh basal) dengan termometer setiap hari.</li>
<li> Mempertimbangkan untuk mendaftarkan hasil pengukuran secara manual di aplikasi 'Perawatan Kesehatan' segera setelah setiap pengukuran dengan termometer biasa.</li>
</ul>
<h3 id="オプション"> Opsi.</h3>
<ul>
<li> Mode suhu tubuh basal</li>
<li> Mode dua tempat desimal</li>
<li> Berfungsi untuk menyelesaikan proses secara otomatis hanya dengan memasukkan angka</li>
</ul>
<h3 id="その他"> Lainnya.</h3>
<ul>
<li> Dapat juga digunakan dengan Apple Watch.</li>
<li> Peluncuran aplikasi Apple Health Care dengan satu ketukan dari dalam aplikasi ini.</li>
<li> Pembatalan pendaftaran dengan satu ketukan segera setelah pendaftaran data.</li>
<li> Unit<ul>
<li> °C Celcius</li>
<li> ℉ Fahrenheit</li>
</ul></li>
</ul>
<h3 id="注意"> Catatan.</h3>
<p> Aplikasi ini tidak dapat membaca, melihat, atau mengelola data sebelumnya pada aplikasi 'Health Care'. Aplikasi ini hanya ditujukan untuk mendaftarkan data di aplikasi 'Health Care'. Silakan periksa data yang didaftarkan pada aplikasi 'Health Care'.</p>
<h3 id="背景"> Latar belakang.</h3>
<p> Orang sering mengukur suhu tubuh mereka (dan suhu tubuh basal) setiap hari dengan menggunakan termometer. Banyak pengguna iPhone mendaftarkan data suhu (atau suhu tubuh basal) mereka pada aplikasi 'Health Care'.</p>
<p> Solusi terbaik adalah dengan menggunakan termometer pintar yang bekerja dengan aplikasi Health Care dan secara otomatis menyimpan pengukuran, tetapi ini bisa jadi mahal dan tidak dapat diakses. Data juga dapat didaftarkan secara manual pada aplikasi 'Health Care', tetapi sayangnya aplikasi 'Health Care' tidak dirancang untuk pendaftaran data pengukuran secara manual secara rutin. Oleh karena itu, memasukkan data secara manual yang terjadi secara terus menerus setiap hari, seperti pengukuran suhu tubuh, dapat menjadi pengalaman yang sangat menyita waktu dan membuat stres.</p>
<p> Aplikasi ini dikembangkan untuk mengatasi masalah tersebut.</p>
